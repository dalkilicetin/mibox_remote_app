 Mi Box Remote — Swift iOS Rewrite Requirements

  1. Uygulama Genel Yapısı

  İki bağımsız protokol paralel çalışır:
  - APK Protokolü — TV'de AirCursor APK kuruluysa aktif. Cursor kontrolü sağlar.
  - ATV Remote v2 — Her TV'de çalışan resmi protokol. Tuş komutları gönderir.

  Her iki bağlantı bağımsızdır. UI her ikisinin durumunu ayrı gösterir.

  4. SetupView Requirements

  4.1 Cihaz Keşif Stratejisi (3 adım sıralı)

  Adım 0 — UDP Broadcast (APK keşfi, paralel başlar)
  - UDP socket aç, 9877 portuna broadcast gönder
  - Magic string: "AIRCURSOR_DISCOVER" (UTF-8)
  - Broadcast adresleri: tüm aktif IPv4 interface'lerin /24 broadcast'i
  - Gelen JSON yanıtında service == "aircursor" kontrolü
  - Yanıttan: ip, cursorPort, atvPairingPort, atvRemotePort, w, h oku
  - Timeout: 2 saniye

  Adım 1 — mDNS (kibar, hızlı)
  - _androidtvremote._tcp ve _androidtvremote2._tcp servislerini dinle
  - Bulunan her servisin host ve port bilgisini al
  - 3 saniye bekle, sonra durdur
  - Aynı IP zaten listede varsa ekleme

  Adım 2 — TCP Port Sweep (mDNS bulamadıysa)
  - Yalnızca _devices boşsa başlat
  - Tüm IPv4 interface'lerden /24 subnet hesapla
  - 1–254 arası her IP için port 6467'ye 500ms timeout ile TCP bağlantı dene
  - 40'lı batch'ler halinde paralel çalış

  4.2 Cihaz Listesi (DiscoveredDevice modeli)

  struct DiscoveredDevice {
      let ip: String
      var hasCert: Bool       // Keychain'de sertifika var mı
      var hasApk: Bool        // UDP'de bulundu mu
      var pairingPort: Int    // default 6467
      var remotePort: Int     // default 6466
  }

  - Keychain'de atv_cert_{ip} anahtarı varsa hasCert = true
  - UDP'de bulunan IP'ler hasApk = true
  - Kart rengi: hasCert → yeşil border, değilse mavi border

  4.3 Cihaz Kartı UI

  - Sol: TV ikonu (hasCert → yeşil, değilse kırmızı)
  - Başlık: "Mi Box"
  - Alt başlık: IP adresi
  - Badge 1: "✓ Eşleştirildi" (yeşil) / "Eşleştirilmedi" (sarı)
  - Badge 2: "APK ✓" (yeşil) / "APK yok" (gri)
  - Sağda: hasCert ise "yeniden eşleştir" butonu + "Bağlan"/"Eşleştir" butonu

  4.4 Alt Butonlar

  - Yeniden Tara: scan sıfırla, yeniden başlat
  - Manuel IP: dialog aç, IP gir, bağlan

  4.5 Manuel IP Dialogu

  - Son kullanılan IP UserDefaults'tan yükle
  - IP geçerliyse APK varlığını 9876 portuna 2 sn timeout ile TCP bağlantı deneyerek kontrol et
  - pairingPort ve remotePort'u UserDefaults'tan oku (yoksa default değerler)

  4.6 Bağlanma Akışı

  Cihaza tıkla
    → Keychain'de cert var mı?
        Evet → RemoteView'a geç
        Hayır → PairingView'ı aç (sheet veya push)
                 → Pairing başarılı → RemoteView'a geç

  5. PairingView Requirements

  5.1 Genel Akış

  1. RSA 2048 keypair üret (SecKeyGeneratePair)
  2. Self-signed X.509 sertifika oluştur (clientName: "ATV Remote", clientId: "com.mibox.remote")
  3. TLS bağlantısı kur, client cert sun
  4. Handshake mesajlarını gönder
  5. PIN giriş ekranını göster
  6. PIN'i doğrula, sertifikaları Keychain'e kaydet

  5.2 Port Deneme Sırası

  [pairingPort, 6467, 6468, 7676] (tekrar olmasın diye unique liste)

  5.3 TLS Bağlantısı (Swift'te doğru çözüm)

  Network.framework → NWConnection
  NWParameters → TLS
    - sec_protocol_options_set_local_identity() → client cert sun
    - sec_protocol_options_set_verify_block() → server cert kabul et (onBadCertificate=true karşılığı)
    - server cert'i bu callback'ten al ve kaydet

  5.4 Pairing Protokol Mesajları

  Pairing Request (binary, sabit):
  [8, 2, 16, 200, 1, 82, <innerLen>, 10, <serviceLen>, ...serviceBytes, 18, <clientLen>, ...clientBytes]
  - service = "ATV Remote"
  - client = "com.mibox.remote"

  Options (sabit): [8, 2, 16, 200, 1, 162, 1, 8, 10, 4, 8, 3, 16, 6, 24, 1]

  Configuration (sabit): [8, 2, 16, 200, 1, 242, 1, 8, 10, 4, 8, 3, 16, 6, 16, 1]

  Mesaj frame formatı: [length_byte, ...payload] (tek byte uzunluk prefix)

  Akış: PairingRequest gönder → ack bekle → Options gönder → ack bekle → Configuration gönder → ack bekle → PIN bekleniyor

  5.5 PIN Doğrulama

  PIN 6 karakter hex string:
  - checkByte = Int(pin[0..<2], radix: 16)
  - pinHashPart = Data(hexString: pin[2...])

  Hash hesaplama:
  hashInput = clientModulus + clientExponent + serverModulus + serverExponent + pinHashPart
  secret = SHA256(hashInput)
  - secret[0] == checkByte ise PIN doğru
  - Doğruysa Secret mesajı gönder: [8, 2, 16, 200, 1, 98, 34, 10, 32, ...secret]
  - Ack bekle → başarılı

  Modulus temizleme: BigInteger'dan bytes alırken başındaki 0x00 byte'ı at (big-endian, no leading zeros)

  5.6 Keychain Kayıt

  - atv_cert_{ip} → PEM string (cert)
  - atv_key_{ip} → PEM string (private key)
  - kSecAttrAccessible = kSecAttrAccessibleWhenUnlocked

  5.7 UI

  - Durum mesajı (orta ekran)
  - Spinner (bağlanırken)
  - 6 haneli PIN text field (büyük, merkez)
  - Onayla butonu
  - Debug log listesi (son 200 satır, renk kodlu: hata=kırmızı, başarı=yeşil, info=gri)

  7. AirMouseView Requirements

  7.1 Mod Toggle Butonu

  - "Kumanda Modu" (kırmızı border, siyah bg) ↔ "Air Modu" (kırmızı bg)
  - Toggle basınca 300ms sonra showCursor() / hideCursor() gönder
  - Toggle anında filter state'i sıfırla

  7.2 Sensör Füzyonu

  Pitch (Y hareketi) — CMAccelerometer:
  beta = atan2(y, sqrt(x² + z²)) * 180/π

  Yaw (X hareketi) — CMMagnetometer:
  alpha = atan2(y, x) * 180/π   (0–360 normalize)

  Low-pass filter (LP = 0.5):
  fbeta = 0.5 * rawBeta + 0.5 * fbeta

  Delta yaw (circular wrap-around):
  rawDa = lastRawAlpha - rawAlpha
  if rawDa > 180:  rawDa -= 360
  if rawDa < -180: rawDa += 360
  filteredDa = 0.5 * rawDa + 0.5 * filteredDa

  Deadzone:
  - pitch: |db| < 0.3 → sıfırla
  - yaw: |da| < 0.05 → sıfırla

  Dinamik hız artışı:
  speed = sqrt(db² + da²)
  boost = speed > 3 ? 1.0 + (speed - 3) * 0.3 : 1.0

  Piksel hareketi:
  dx = round(da * boost * 25)
  dy = round(db * sensitivity/25 * boost * -25)

  Rate limit: 16ms (60 Hz)

  7.3 TIKLA Alanı (büyük kırmızı alan)

  - onPanStart → start pos, start time kaydet
  - onPanUpdate → dx/dy biriktir:
    - accumV >= 65 → sendKey(20/19) + setScrollMode(1) + sıfırla
    - accumH >= 65 → sendKey(22/21) + setScrollMode(2) + sıfırla
  - onPanEnd:
    - moved < 12px && elapsed < 250ms → tap!
        - Air modda: APK tap + ATV Enter (keyCode 23)
      - Kumanda modda: ATV Enter

  7.4 Swipe Scrollbar (sağ dar şerit, kırmızı border)

  - Dikey sürükleme → 40px threshold → sendSwipe()
  - Yukarı/aşağı ikonları → direkt sendSwipe()
  - sendSwipe: cursor pozisyonundan 150px yukarı/aşağı, duration=100ms

  7.5 Butonlar

  - Geri (Back): keyCode 4
  - Home: keyCode 3
  - Ses+: keyCode 24
  - Ses-: keyCode 25

  7.6 Klavye Popup (bottom sheet)

  11. APK Servisi (MiBoxService eşdeğeri)

  Bağlantı

  - TCP Socket, port 9876
  - Timeout: 5 saniye
  - Gelen JSON satırlarını parse et: x, y, atvPairingPort, atvRemotePort
  - Disconnect'te stream'e false gönder

  Komutlar (JSON over TCP, \n ile biter)

  ┌───────────────┬─────────────────────────────────────────────────────────────────────┐
  │     Komut     │                                JSON                                 │
  ├───────────────┼─────────────────────────────────────────────────────────────────────┤
  │ moveCursor    │ {"type":"move","dx":dx,"dy":dy}                                     │
  ├───────────────┼─────────────────────────────────────────────────────────────────────┤
  │ tap           │ {"type":"tap"}                                                      │
  ├───────────────┼─────────────────────────────────────────────────────────────────────┤
  │ sendKey       │ {"type":"key","code":keyCode}                                       │
  ├───────────────┼─────────────────────────────────────────────────────────────────────┤
  │ sendText      │ {"type":"text","value":"..."}                                       │
  ├───────────────┼─────────────────────────────────────────────────────────────────────┤
  │ hideCursor    │ {"type":"hide"}                                                     │
  ├───────────────┼─────────────────────────────────────────────────────────────────────┤
  │ showCursor    │ move to center + {"type":"show"}                                    │
  ├───────────────┼─────────────────────────────────────────────────────────────────────┤
  │ setScrollMode │ {"type":"scroll_mode","mode":1} + 500ms sonra mode=0                │
  ├───────────────┼─────────────────────────────────────────────────────────────────────┤
  │ sendSwipe     │ {"type":"swipe","x1":...,"y1":...,"x2":...,"y2":...,"duration":...} │
  └───────────────┴─────────────────────────────────────────────────────────────────────┘

  Sabitler

  - SCREEN_W = 1920, SCREEN_H = 1080
  - Cursor başlangıç: (960, 540) (merkez)
  - cursorX, cursorY → moveCursor'dan güncellenir, clamp: 0–SCREEN_W / 0–SCREEN_H

  12. ATV Remote Servisi (AtvRemoteService eşdeğeri)

  Bağlantı

  - Network.framework NWConnection, port 6466
  - TLS + client certificate (Keychain'den)
  - Server cert'i kabul et (pinning yok)

  Mesaj Formatı

  Varint length prefix: her mesajdan önce varint encode edilmiş byte sayısı

  Varint encode: 7 bit chunk, MSB=1 devam var, MSB=0 son byte

  Protobuf Encoder

  func writeVarint(field: Int, value: Int)   // wire type 0
  func writeString(field: Int, value: String) // wire type 2
  func writeBytes(field: Int, data: Data)     // wire type 2

  Gelen Mesajları İşle

  ┌──────┬──────────────────┬───────────────────────────────┐
  │ Tag  │      Mesaj       │            Aksiyon            │
  ├──────┼──────────────────┼───────────────────────────────┤
  │ 0x0A │ remote_configure │ configure + set_active gönder │
  ├──────┼──────────────────┼───────────────────────────────┤
  │ 0x42 │ ping_request     │ pong gönder                   │
  ├──────┼──────────────────┼───────────────────────────────┤
  │ 0x4A │ ping_response    │ yoksay                        │
  ├──────┼──────────────────┼───────────────────────────────┤
  │ 0x12 │ set_active ack   │ handshake tamamlandı          │
  └──────┴──────────────────┴───────────────────────────────┘

  configure Mesajı

  RemoteMessage { remote_configure=1: { code1=615, device_info { unknown1=1, unknown2="1", package_name="com.mibox.remote", app_version="1.0.0" } } }

  set_active Mesajı

  RemoteMessage { remote_set_active=2: { active=615 } }

  Key Inject

  RemoteMessage { remote_key_inject=10: { key_code=1: keyCode, direction=2: 3 } }
  - SHORT = 3
  - START_LONG = 1, END_LONG = 2
